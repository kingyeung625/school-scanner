## 📋 功能說明

### 篩選功能

1. **學校名稱搜索**：在側邊欄頂部輸入學校名稱
2. **區域**：選擇學校所在區域
3. **校網**：選擇小一學校網
4. **辦學團體**：選擇辦學團體（按學校數量排序）
5. **資助類型**：資助、官立、私立、直資
6. **學生性別**：男女、男、女
7. **宗教**：基督教、天主教、佛教等
8. **教學語言**：中文、中英文等
9. **關聯學校**：一條龍、直屬、聯繫
10. **課業安排**：下午導修、小一不設測考等
11. **學校特色**：通過標籤或關鍵詞搜索

### 使用流程

1. **搜索學校**：
   - 在側邊欄輸入學校名稱或選擇篩選條件
   - 點擊篩選條件展開/摺疊

2. **查看結果**：
   - 主頁面顯示符合條件的學校列表
   - 每個學校卡片顯示基本信息

3. **比較學校**：
   - 勾選學校卡片上的 "比較" 複選框（最多 4 所）
   - 點擊 "比較已選" 按鈕查看比較視圖

4. **查看詳細信息**：
   - 點擊 "詳細資料" 按鈕查看學校完整信息
   - 使用標籤頁切換不同類別的信息

5. **切換語言**：
   - 點擊頂部的 "繁體/簡體" 單選按鈕切換語言

## 🐛 故障排除

### 問題：應用無法啟動

**解決方案**：
1. 確認已安裝所有依賴：`pip install -r requirements.txt`
2. 確認 Python 版本 >= 3.8
3. 檢查 CSV 文件路徑是否正確

### 問題：找不到 CSV 文件

**解決方案**：
應用會自動嘗試多個路徑：
- `../attached_assets/database_school_info_1763020452726.csv`
- `attached_assets/database_school_info_1763020452726.csv`
- `data/database_school_info_1763020452726.csv`

如果都不存在，請檢查文件位置並調整 `app.py` 中的路徑。

### 問題：篩選結果為空

**解決方案**：
1. 確認已輸入搜索條件或選擇篩選器
2. 嘗試清除所有篩選並重新選擇
3. 檢查篩選條件是否過於嚴格

### 問題：應用運行緩慢

**解決方案**：
- 首次加載會較慢（需要解析 CSV）
- 後續訪問會使用緩存，速度更快
- 如果仍然很慢，可以考慮優化 CSV 文件大小

## 📦 部署到 Streamlit Cloud

1. 將代碼推送到 GitHub

3. 使用 GitHub 登錄
4. 點擊 "New app"
5. 選擇倉庫和分支
6. 設置主文件路徑為 `streamlit_app/app.py`
7. 點擊 "Deploy"
